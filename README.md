## 📋 프로젝트 개요

이 프로젝트는 클래식 아케이드 게임인 스페이스 인베이더를 Python으로 구현한 것입니다. 게임 개발 학습을 위한 튜토리얼 프로젝트로 제작되었습니다.

## 🎮 게임 설명

스페이스 인베이더는 플레이어가 우주선을 조종하여 화면 상단에서 내려오는 외계인들을 격추하는 슈팅 게임입니다.

## 📁 프로젝트 구조

```
python-game/
├── assets/          # 게임 리소스 (이미지, 사운드 등)
├── example/         # 예제 코드
├── objects/         # 게임 오브젝트 클래스
├── utils/           # 유틸리티 함수
├── constants.py     # 게임 상수 정의
└── main.py          # 메인 실행 파일
```

### 주요 파일 설명

- **main.py**: 게임의 메인 루프와 초기화를 담당하는 진입점
- **constants.py**: 게임에서 사용되는 각종 상수값 정의 (화면 크기, 속도 등)
- **objects/**: 플레이어, 적, 총알 등 게임 오브젝트 클래스들
- **utils/**: 충돌 감지, 점수 계산 등 유틸리티 함수들
- **assets/**: 게임에 사용되는 이미지, 사운드 파일들

## 🚀 실행 방법

### 필수 요구사항

- Python 3.x
- Pygame 라이브러리

### 설치

1. 저장소 클론
```bash
git clone https://github.com/icemokacat/python-game.git
cd python-game
```

2. 필요한 패키지 설치
```bash
pip install pygame
```

### 게임 실행

```bash
python main.py
```

## 🎯 게임 조작법

- **←/→ 방향키**: 우주선 좌우 이동
- **스페이스바**: 발사
- **ESC**: 게임 종료
